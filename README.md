# Map URL data extractor

1. Install packages
```
npm install 
```
2. Create .env and add following values

```
FIREBASE_API_KEY=
FIREBASE_AUTH_DOMAIN=
FIREBASE_PROJECT_ID=
FIREBASE_STORAGE_BUCKET=
FIREBASE_MESSAGING_SENDER_ID=
FIREBASE_APP_ID=
FIREBASE_MEASUREMENT_ID=
```
3. Compile the TypeScript code:
```
npx tsc
```

4. Run script

```
node .\src\index.js
```
